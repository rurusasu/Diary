# PIL/Pillow について

### Image モード一覧

次のスライドに image モードの一覧を示す。

---

| モード | 説明                              |
| ------ | --------------------------------- |
| 1      | 1bit マスクに使用 論理演算が可能  |
| L      | 8bit グレースケール               |
| P      | パレットモード                    |
| RGB    | 8 bit × 3                         |
| RGBA   | 8 bit × 4                         |
| RGBa   | アルファチャンネルで RGB 値を乗算 |
| CMYK   | 8 bit × 4 印刷関連でよく使われる  |
| YCbCr  | 8bit × 3 ビデオ関連でよく使われる |
| HSV    | 8 bit × 3 pillow のみ             |
| LA     | アルファチャンネルで L 値を乗算   |
| I      | 32 bit 整数                       |
| F      | 32bit 浮動小数                    |
